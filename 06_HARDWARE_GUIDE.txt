SOVEREIGN AI STARTER PACK — OLLAMA EDITION
06 — Hardware Guide

------------------------------------------------------------
THE SHORT ANSWER
------------------------------------------------------------

You probably already have hardware that works. If your computer runs
modern macOS, Windows 10/11, or Linux and has 8 GB RAM, you can start
today with a small model.

Better hardware = faster replies and access to smarter models. It is not
all-or-nothing.


------------------------------------------------------------
MINIMUM VIABLE SETUP
------------------------------------------------------------

  RAM:        8 GB
  Disk:       15 GB free (10 GB for OS headroom + one 7B model)
  GPU:        Not required
  OS:         macOS 12+, Windows 10/11 64-bit, or modern Linux
  Internet:   Required once for install and model download

Good first model: phi3, mistral, or llama3.2:1b


------------------------------------------------------------
RECOMMENDED SETUP (DAILY USE)
------------------------------------------------------------

  RAM:        16 GB
  Disk:       30 GB free (room for 3–5 models)
  GPU:        Apple Silicon Mac OR NVIDIA with 8 GB+ VRAM
  OS:         Same as above

Good daily models: llama3.2, qwen2.5, mistral, gemma2


------------------------------------------------------------
ENTHUSIAST / POWER USER
------------------------------------------------------------

  RAM:        32–64 GB
  Disk:       100 GB+ if collecting large models
  GPU:        NVIDIA 12–24 GB VRAM (RTX 3060 12GB, 4070, 4080, etc.)
              or Apple Silicon Mac with 24 GB+ unified memory
  OS:         Linux or Windows often chosen for NVIDIA flexibility

Can run: 13B–34B models comfortably; 70B with tradeoffs


------------------------------------------------------------
APPLE SILICON MAC (M1 / M2 / M3 / M4)
------------------------------------------------------------

One of the best beginner platforms for local AI.

  8 GB unified memory:
    Stick to 3B–7B models. One model at a time. Close other apps.

  16 GB unified memory:
    Sweet spot. llama3.2, qwen2.5, mistral run well for daily chat.

  24 GB+ unified memory:
    Room for larger models and longer context. Strong daily driver.

Tips:
  • Ollama uses Apple's Metal acceleration automatically
  • Fan noise under load is normal
  • Monitor memory pressure in Activity Monitor if sluggish


------------------------------------------------------------
WINDOWS PC
------------------------------------------------------------

  CPU-only (no NVIDIA GPU):
    Works like any laptop — use smaller models, expect slower speed.

  NVIDIA GPU:
    Ollama uses CUDA automatically when drivers are installed.
    Check VRAM in Task Manager → Performance → GPU.

  VRAM guide:
    6 GB   — 7B models, tight fit
    8 GB   — 7B–8B comfortable
    12 GB  — 13B possible
    24 GB  — 34B and some 70B quantizations

  AMD GPU:
    Support varies. CPU inference or check latest Ollama docs for ROCm.


------------------------------------------------------------
LINUX
------------------------------------------------------------

  Popular choice for power users and homelabs.
  NVIDIA + CUDA is the most straightforward GPU path.
  CPU-only installs work fine for small models.
  Great for always-on local AI on a desktop or mini PC.


------------------------------------------------------------
MINI PC / HOMELAB / OLD HARDWARE
------------------------------------------------------------

An old desktop or mini PC with 16 GB RAM can be a dedicated local AI box:

  • Install Linux or Windows
  • Install Ollama
  • Run smaller models 24/7 without tying up your laptop
  • Keep it on your local network only (no port forwarding unless you
    know what you are doing)

Good for: family chat bot, offline FAQ assistant, experimentation.


------------------------------------------------------------
LAPTOP VS DESKTOP
------------------------------------------------------------

  Laptop:
    Portable, quieter at idle, but thermals limit long sessions.
    Best for learning and casual use.

  Desktop:
    Better cooling, easier GPU upgrades, more RAM slots.
    Best for heavy models and long runs.


------------------------------------------------------------
WHAT TO UPGRADE FIRST
------------------------------------------------------------

Priority order for most people:

  1. RAM (8 → 16 GB is the biggest jump for beginners)
  2. Disk space (NVMe SSD if still on a slow drive)
  3. GPU (if on Windows/Linux and you want speed)
  4. Whole system (if CPU is 10+ years old)

You do not need to buy anything to start. Run phi3 on what you have.


------------------------------------------------------------
THERMAL AND POWER NOTES
------------------------------------------------------------

  • Local AI uses CPU/GPU hard — fans will spin up
  • Laptops on soft surfaces can throttle — use a hard desk
  • Long sessions on battery drain fast — plug in when possible
  • If the machine gets uncomfortably hot, stop the model (Ctrl+C)
    and let it cool


------------------------------------------------------------
NETWORK — YOU DO NOT NEED A SERVER
------------------------------------------------------------

Everything in this pack runs on one machine. You do not need:

  • A cloud VPS
  • A NAS (unless you choose to store files there yourself)
  • Port forwarding for basic use

Open WebUI at localhost:3000 is only reachable from YOUR computer unless
you deliberately expose it.


------------------------------------------------------------
CHECK WHAT YOU HAVE (BEFORE BUYING ANYTHING)
------------------------------------------------------------

  macOS:     Apple menu → About This Mac → Memory
  Windows:   Settings → System → About
  Linux:     run  free -h  in terminal

Compare to 03_MODEL_HORSEPOWER_GUIDE.txt and pick a model that fits.
