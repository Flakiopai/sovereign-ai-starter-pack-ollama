ABOUT THIS PACK
------------------------------------------------------------

Created by MATTEBLACK STUDIOS as a sovereign, local-first starter guide
for running open-source language models on your own hardware.

This pack documents tools and practices. The software below is third-party,
free or open-source, and must be downloaded separately by the user. Nothing
in this pack implies endorsement, affiliation, or partnership with any
listed project unless explicitly stated.


------------------------------------------------------------
PRIMARY TOOLS (DOCUMENTED IN THIS PACK)
------------------------------------------------------------

OLLAMA
  Role:    Local AI engine — downloads and runs language models
  Site:    https://ollama.com
  License: MIT (verify on official repository)
  Note:    Primary tool for this starter pack

OPEN WEBUI
  Role:    Optional browser chat interface for Ollama
  Site:    https://github.com/open-webui/open-webui
  License: Open-source (see repository for current license)
  Note:    Self-hosted; runs on user's machine

DOCKER DESKTOP
  Role:    Container platform (optional, for Open WebUI install)
  Site:    https://www.docker.com
  Note:    Free tier available; not required for Ollama alone
